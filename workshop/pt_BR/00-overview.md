# 🎮 VS Code GitHub Copilot Agent Lab

[📚 Guia do Lab](https://copilot-dev-days.github.io/agent-lab-java/docs/) • [← README](../README.md)

---

> **Duração:** ~1 hora
> **Nível:** Intermediário
> **Stack:** Java 21 / Spring Boot 3.4.2 / Apache Maven

Bem-vindo ao Soc Ops — um workshop prático onde você vai transformar um app simples de Social Bingo em algo incrível usando o Agent Mode do VS Code com GitHub Copilot.

---

## 📋 Lista Rápida

Antes de começar, verifique:

- [ ] VS Code **v1.107+** (sem atualizações pendentes)
- [ ] Logado com **GitHub Copilot** (Pro, Business ou Enterprise)
- [ ] Git instalado
- [ ] Java 21 JDK instalado
- [ ] Painel de Chat aberto e Agente pronto

> 💡 **Dica:** Use o DevContainer para um ambiente pré-configurado!

---

## 🎯 O que Você Vai Aprender

| # | Habilidade | Descrição |
|---|------------|-----------|
| 1 | **Engenharia de Contexto** | Ensinar a IA sobre seu codebase com instruções |
| 2 | **Primitivos Agênticos** | Usar agentes em segundo plano, agentes na nuvem e workflows personalizados |
| 3 | **Desenvolvimento Design-First** | Deixar a IA iterar na UI enquanto você guia a visão |
| 4 | **Desenvolvimento Orientado a Testes** | Usar agentes TDD para desenvolvimento confiável de funcionalidades |

---

## 📚 Partes do Lab

| Parte | Título | Tempo | Descrição |
|-------|--------|-------|-----------|
| [**01**](01-setup.md) | Configuração & Engenharia de Contexto | 15 min | Clonar, configurar e ensinar a IA sobre seu projeto |
| [**02**](02-design.md) | Frontend Design-First | 15 min | Redesign da UI com temas criativos |
| [**03**](03-quiz-master.md) | Quiz Master Personalizado | 10 min | Criar seus próprios temas de quiz com agentes personalizados |
| [**04**](04-multi-agent.md) | Desenvolvimento Multi-Agente | 20 min | Construir novas funcionalidades com TDD e agentes de design |

---

## 💡 Dicas

1. **Mantenha o navegador aberto** — Veja atualizações ao vivo enquanto programa
2. **Faça commit com frequência** — Salve estados funcionando regularmente
3. **Use checkpoints** — Reverta mudanças inesperadas com Checkpoints & Desfazer do chat
4. **📌 Fixe este guia** — Mantenha-o visível enquanto trabalha
